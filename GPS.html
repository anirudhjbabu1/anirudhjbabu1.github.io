Embodied AI / Transformers

Latency: < 45ms Inference

This project focuses on bridging high-level semantic reasoning with low-level robotic control. By utilizing Vision-Language-Action (VLA) models, I enabled a 7-DOF arm to interpret natural language commands within dynamic, unstructured environments.

GPS-Denied Autonomy

SLAM / VIO / Path Planning

Hardware: OAK-D, Jetson Orin
Algo: Fast-LIO2 / VINS-Mono

Developed a robust navigation stack for a quadrotor in confined indoor environments. The system utilizes Visual-Inertial Fusion to maintain localization during high-dynamic maneuvers where GPS signals are completely absent.
